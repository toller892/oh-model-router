# oh-model-router

本地模型路由和管理器。智能选择本地或云端模型。

## 功能特性

- 🔀 智能路由：自动选择最佳模型
- 💰 成本优化：优先使用本地模型
- 🔄 自动降级：本地失败时切换云端
- 📊 性能追踪：监控速度、质量、成本
- 🏠 本地优先：通过 Ollama 运行本地模型
- ⚙️ 简单配置：YAML/JSON 配置

## 使用方法

### 路由查询

```
用模型路由器处理："解释量子计算"
```

### 检查状态

```
显示模型路由器状态
```

### 基准测试

```
运行模型路由器基准测试
```

## 路由策略

考虑因素：
- 任务复杂度
- 模型可用性
- 成本约束
- 性能要求
- 隐私级别

## 配置

```json
{
  "local": {
    "provider": "ollama",
    "models": ["qwen2.5", "llama3"]
  },
  "routing": {
    "strategy": "cost-optimized",
    "localFirst": true
  }
}
```

## 依赖

- Node.js >= 18
- Ollama（本地模型）
- 最少 8GB 内存
